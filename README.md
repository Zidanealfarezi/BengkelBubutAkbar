# Bengkel Bubut Akbar Landing Page

Landing page statis satu halaman untuk promosi Bengkel Bubut Akbar dengan desain industrial berbasis Tailwind CDN.

## Cara membuka

Buka file `index.html` langsung di browser.

## Yang perlu diganti sebelum dipublikasikan

- Nomor WhatsApp: ubah `WHATSAPP_NUMBER` di `script.js`.
- Alamat workshop: ubah link Google Maps di `index.html`.
- Foto: ganti URL gambar ilustrasi pada hero dengan foto asli bengkel.

## File utama

- `index.html`: struktur halaman dan konten.
- `script.js`: link WhatsApp otomatis.
